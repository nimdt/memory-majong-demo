How to play

  • Pick two matching free tiles to remove them.
  • A tile is free when nothing sits on top of it and at least one side edge is open.
  • Memory Mahjong is the default start mode, with standard Mahjong available as the toggle.
  • Use Hint if you want one free matching pair highlighted.
  • Keyboard shortcuts: N new layout, H hint, R reshuffle, M toggle memory mode, 1/2/3 switch layouts.
